# Proyecto Tienda de Sneakers - POO

## Descripción
Este proyecto consiste en un sistema de tienda virtual desarrollado en Python utilizando Programación Orientada a Objetos.

El sistema permite:
- Mostrar productos
- Agregar productos al carrito
- Realizar compras
- Procesar pagos
- Calcular envíos
- Guardar información en archivos JSON

## Tecnologías usadas
- Python
- JSON
- GitHub

## Conceptos POO aplicados
- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Métodos

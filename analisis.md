# Análisis del Proyecto

## Problema
Muchas tiendas necesitan organizar productos, ventas y clientes de forma digital.

## Solución
Se desarrolló un sistema en Python que permite administrar productos y realizar compras mediante un menú interactivo.

## Objetivo General
Desarrollar una tienda virtual usando Programación Orientada a Objetos.

## Objetivos Específicos
- Implementar clases y objetos
- Aplicar herencia y polimorfismo
- Utilizar manejo de archivos JSON
- Simular compras y pagos

## Clases Utilizadas
- Usuario
- Cliente
- Producto
- Carrito
- Pedido
- Pago
- Envio
- Tienda

## Herencia
La clase Cliente hereda de Usuario.

## Polimorfismo
El método mostrar_info() fue sobrescrito en Cliente.

## Manejo de Archivos
Los productos se almacenan en un archivo JSON.
